# Chrome开发者工具

## 性能相关的面板
- 网络面板
- `Performance`面板
- 内存面板

### 网络面板
- 控制器
  1. 红色圆点：开始/暂停抓包
  2. 全局搜索
  3. `Disabled Cache`
  4. `Online`按钮
- 过滤器: 根据文件类型过滤下载内容
- 抓图信息: 用户等待页面加载时间内所看到的内容
- 时间线: 用以展示`HTTP`, `HTTPS`, `WebSocket`加载的状态和时间的一个关系
- 详细列表: 每个资源从发起请求到完成请求的中间过程状态
- 下载信息概要
  - `DOMContentLoad`: 该事件发生后，页面已经构建好`DOM`，此时构建`DOM`所需的`HTML`, `JavaScript`及`CSS`文件都已经下载完成
  - `Load`: 说明浏览器已加载所有的资源（图像、样式表）
