### 总结一些项目中可做的性能优化
+ 减少`HTTP`请求数
+ 减少`DNS`查询
+ 使用`CDN`
+ 避免重定向
+ 图片懒加载
+ 减少`DOM`元素数量
+ 减少`DOM`操作
+ 使用外部的`JavaScript`和`CSS`
+ 压缩`JavaScript`、`CSS`、字体、图片等
+ 优化`CSS Sprite`
+ 使用`iconfont`
+ 字体裁剪
+ 多域名分发划分内容到不同域名
+ 尽量减少`iframe`的使用
+ 避免图片`src`为空
+ 把样式表放在头部
+ 把脚本放在页面底部

### SEO优化
+ 合理使用`title`, `description`, `Keywords`(`description`与`Keywords`为`<meta>`标签)
